1. El principio del Tallo Cerebral.
    A diferencia de los sistemas comunes donde el sensor le dice al programa que hacer,
    aquí el tallo cerebral es el dueño absoluto del tiempo.
        - Nada ocurre si el Ritmo no lo autoriza.
        - Esto garantiza Determinismo: el sistema se comporta igual en una Canaima(Classmate)
        que en una Supercomputadora, porque el tiempo es una constante sagrada, no una
        variable del sistema operativo.

2. El triángulo de la Autorregulación (Nervio -> Tálamo -> Hipotálamo)
    - El nervio no entrega un dato, entrega una "intensidad"(señal normalizada).
    - El Tálamo evalúa la salud de esa señal. Si detecta "silencio" o "caos",
    etiqueta la señal como inválida.
    - El Hipotálamo recibe esa señal etiquetada. Si es inválida, ignora a la Prefrontal y
    activa el "Protocolo de ceguera": un estado de defensa pre-programado para mantener con
    vida a los animales o estables los elementos, mientras recupera el sentido.

3. Desacoplamiento de Propósito (El Mundo Mental)
    La Corteza Asistente (PID) y el Hipotálamo no saben que existen animales o elementos.
    Ella solo sabe que hay un Mundo Mental que pide equilibrio.
        - Esto permite que este mismo cerebro sea usado en la Industria Petrolera
        simplemente cambiando la carpeta 3_Mundo_Mental. El "cerebro" sigue siendo el mismo;
        solo cambia lo que "imagina" y "siente".

4. La Jerarquía del Veto (Disciplina)
    En este sistema, la inteligencia (PID) es un asistente, no el jefe.
        - La Prefrontal sugiere: "Bajemos la potencia al 20% para ahorrar luz".
        - El Hipotálamo puede responder: "Veto la orden. La frecuencia de calor es
        demasiado alta, mantendremos 100% de potencia por instinto de supervivencia".
        - Esta jerarquía asegura que unn error en el código complejo(PID) nunca mate o dañe
        la producción.
